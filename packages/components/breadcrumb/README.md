## Nunjucks macro

```html
{% from 'components/breadcrumb/macro.njk' import breadcrumb %}

{{ breadcrumb({
  items: [
    {
      URL: "https://www.nhs.uk/",
      label: "Home"
    },
    {
      URL: "https://www.nhs.uk/conditions",
      label: "Health A-Z"
    }
  ],
  parentURL: "https://www.nhs.uk/conditions/abscess/",
  parentLabel: "Abscess"
}) }}
```
